# 团队工具：一键安装法律 AI 插件套件

> 通过我们 VPS 国内中转，零翻墙、零代理。
> 一条命令拿到 12 个法律业务领域插件。

## 二、前置条件

- 已经按 `VPS-团队工具分发-教程.md` 装好 **Claude Code**（`claude --version` 能输出版本号）
- Claude Code 版本 ≥ 2.1.0（支持 `/plugin` 命令）

## 三、安装（推荐方式）

打开 Claude Code，输入：

```
/plugin marketplace add https://files.uuhfn.cloud/git/bifrost-internal-plugins.git
```

成功后看到：

```
✓ Added marketplace 'team-uuhfn' (12 plugins)
```

然后按需 install：

```
/plugin install commercial-legal@team-uuhfn
/plugin install corporate-legal@team-uuhfn
/plugin install employment-legal@team-uuhfn
```

或律所典型套餐（建议全员装）：

```
/plugin install commercial-legal@team-uuhfn
/plugin install corporate-legal@team-uuhfn
/plugin install employment-legal@team-uuhfn
/plugin install litigation-legal@team-uuhfn
/plugin install privacy-legal@team-uuhfn
```

AI 业务方向再加：

```
/plugin install ai-governance-legal@team-uuhfn
/plugin install ip-legal@team-uuhfn
```

## 四、常用命令清单

| 操作 | 命令 |
|---|---|
| 查所有已添加 marketplace | `/plugin marketplace list` |
| 查 marketplace 里的插件 | `/plugin search @team-uuhfn` |
| 安装单插件 | `/plugin install <name>@team-uuhfn` |
| 检查更新 | `/plugin marketplace update team-uuhfn` |
| 卸载插件 | `/plugin uninstall <name>` |
| 卸载整 marketplace | `/plugin marketplace remove team-uuhfn` |

## 五、安装后会发生什么

每个 plugin 安装到 `~/.claude/plugins/team-uuhfn/<plugin-name>/`：

- `agents/` 子代理定义（例如"合同审查员"）
- `skills/` 可调用技能（例如"识别民法典适用条款"）
- `commands/` 自定义 slash 命令（例如 `/contract-review`）
- `hooks/` 钩子脚本（一般不需要管）

启动新会话，`/help` 能看到新增的 slash 命令。

## 六、验证 VPS 链路

如果安装失败先验链路。任一 OS 跑：

```bash
git ls-remote https://files.uuhfn.cloud/git/bifrost-internal-plugins.git
```

应返回一行 commit SHA + `refs/heads/main`。

如果想确认上游插件源也通：

```bash
git ls-remote https://files.uuhfn.cloud/git/claude-for-legal-ZH.git
```

同样应返回 SHA + refs。

## 七、出错时排查

| 症状 | 原因 | 解决 |
|---|---|---|
| `marketplace add` 报 `failed to clone` | git 没装 / PATH 里找不到 | 装 Git for Windows，重启 CMD |
| `marketplace add` 报 SSL 错误 | 时间不对 / 旧根证书 | 同步系统时间，老 Win 跑 Windows Update |
| `marketplace add` 成功但 `install` 报 `path not found` | 上游 mirror 没同步 | 联系管理员跑一次 `Start-ScheduledTask -TaskName claude-for-legal-sync` |
| `/help` 看不到新命令 | 没重启 Claude Code | 退出当前会话，开新会话 |
| `install` 卡在 sparse-checkout | 客户端 git < 2.25 | 升级 Git for Windows 到最新版 |

无法解决，把报错截图发给 ZRainbow。

## 八、隐私与安全说明

- 插件源代码全部在你本地 `~/.claude/plugins/` 下可见，**不调用任何境外服务**
- 实际 AI 推理走你在 cc-switch 中配的服务商（DeepSeek / Anthropic / 阿里 etc）
- 律所敏感数据**是否经过 LLM 取决于你在 Claude Code 里输入什么**，与插件无关
- 上游 claude-for-legal-ZH 是 Apache-2.0 公开仓库；我们 VPS 仅做镜像中转
- VPS 拒绝 push（`receive-pack disabled`），团队成员不会误推坏 marketplace

## 九、更新插件版本

我们 VPS 每天 02:00 自动从上游同步。你想立即拿最新：

```
/plugin marketplace update team-uuhfn
/plugin uninstall commercial-legal
/plugin install commercial-legal@team-uuhfn
```

或简化（如果 plugin.json 没固定 version，按 commit SHA 自动跟随）：

```
/plugin marketplace update team-uuhfn
```
